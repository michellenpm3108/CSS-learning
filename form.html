FORM'S ELEMENTS

  1. <form>- the form itself
  2. <label> - describes the different form elements
  3. <input> - where user can put information. Have many different types (text, color, date, password, email, file, checkbox, radio etc.)
  4. <button> - used to submit the form (among other things)

FORM'S ATTRIBUTES

  1. FORM action="#": when someone clicks the submit button, what is the action? (go to the # page)
  2. FORM method="post": when someone clicks the submit button, what is the form doing? (post = send that information to the server)
  3. INPUT type: text, email, password, color, date, radio, checkbox, file
  4. LABEL - for | INPUT - id: used to connect label with an input
  5. BUTTON type: submit, reset
